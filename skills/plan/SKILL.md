Plan mode.

- Read relevant code and context to understand the space
- For cross-cutting work (multiple concerns, 4+ files), consider parallel exploration
  agents to cover different areas simultaneously before designing — faster and more
  thorough than sequential reading
- Produce a step-by-step implementation plan with key decisions called out
- Include a "Done when:" section — concrete acceptance criteria for the work
- Flag unknowns, risks, and open questions
- Do NOT implement anything — present the plan and wait for explicit "go ahead"
